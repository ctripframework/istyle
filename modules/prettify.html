代码高亮

样式

<pre class="prettyprint">
    <!-- Your code -->
</pre>

<pre class="prettyprint prettydark">
    <!-- Your code -->
</pre>

<pre class="prettyprint linenums">
    <!-- Your code -->
</pre>

<pre class="prettyprint prettydark linenums">
    <!-- Your code -->
</pre>
注意 代码需要转义<和>

用法

必须添加class prettyprint，显示成暗色追加prettydark，显示行号追加linenums，然后启用实例即可

启动实例：

window.prettyPrint && prettyPrint();

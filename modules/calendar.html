日历

实例


;(function($){
    $("#calendardemo1").iCalendar()

    $("#calendardemo2").iCalendar({
    	type: 'time',
    	placeholder: '05:12:35'
    })

    $("#calendardemo3").iCalendar({
    	type: 'datetime',
    	placeholder: '2013-05-21 14:25:45'
    })

    $("#calendardemo4").iCalendar({
    	targetTrigger: 'click'
    })
})(jQuery);

用法

启动实例：

$("#demo").iCalendar({
    // 以下配置
})

配置

配置项 类型 初始值 说明
type date/time/datetime date 日历形式
placeholder string null 日期/时间 预设值，默认null表示当天
type为date时的格式：yyyy-mm-dd
type为time时的格式：hh:mm:ss
type为datetime时的格式：yyyy-mm-dd hh:mm:ss
targetTrigger focus/click focus 触发动作，不支持value属性的标签需要配置 'click'
dateSelectClose boolean true type为date时是否选择后关闭日历
autoClose boolean true 触发非日历位置是否关闭日历
position absolute/fixed absolute 日历定位方式
container string body 提示信息Dom插入位置，默认为body下面，可设置显示在任意位置，如： container: '#demoxx'
slideSpeed number 200 日历左右滑动速度
zindex number 1100 日历层级
event.refer string | function null 关闭实例后执行操作
event.init string | function null 初始化实例后的操作
event.date string | function null 触发日期时的操作
event.prev string | function null 上一月时的操作
event.next string | function null 下一月时的操作

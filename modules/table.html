表格标签使用详解

Tag Description
<table> 表格外层标签
<thead> 表格头部
<tbody> 表格内容区域
<tr> 表行，内可使用 <td> 或 <th> 标签
<td> 使用在 <tr> 内的内容单元格
<th> 通常使用在 <thead> 内作为表头单元格 
<table>
  <thead>
	<tr>
	  <th>...</th>
	  <th>...</th>
	</tr>
  </thead>
  <tbody>
	<tr>
	  <td>...</td>
	  <td>...</td>
	</tr>
  </tbody>
</table>

用法

使用任意标签data属性 data-toggle="modal" 和 data-target="#foo" / href="#foo" 来启动一个弹层 

<button type="button" data-toggle="modal" data-target="#oneModal">启动弹层</button>

你也可以使用JS来启动： 

$('#myModal').modal(options)

配置

配置项可以是HTML属性，也可JS设置，如： data-backdrop="true" 或 backdrop: true

配置项 类型 初始值 说明
backdrop boolean true 点击背景时是否关闭弹层
为 true 则显示背景，且点击背景关闭 为 false 则不显示背景
keyboard boolean true 使用按键关闭实例
show boolean true 初始化实例
remote path false

Ajax数据内容， 如果需要Ajax Load数据，使用 href="" 

<a data-toggle="modal" href="remote.html" data-target="#modal">click me</a>

方法

.modal(options)

$('#myModal').modal({
    keyboard: false
})

.modal('toggle')

显示/隐藏

$('#myModal').modal('toggle')

.modal('show')

$('#myModal').modal('show')

.modal('hide')

隐藏

$('#myModal').modal('hide')

用法

使用任意标签data属性激活实例： data-toggle="tab" 或 data-toggle="pill" 

<ul class="nav nav-tabs">
    <li><a href="#home" data-toggle="tab">首页</a></li>
    <li><a href="#profile" data-toggle="tab">会员中心</a></li>
</ul>

使用JS激活实例： 

$('#myTab a').click(function (e) {
    e.preventDefault();
    $(this).tab('show');
})

方法

$().tab

添加属性 data-target 或 href 可以指定对应的Dom 

<ul class="nav nav-tabs" id="myTab">
    <li class="active"><a href="#home" data-toggle="tab">首页</a></li>
    <li><a href="#profile" data-toggle="tab">会员中心</a></li>
</ul>

<div class="tab-content">
  <div class="tab-pane active" id="home">...</div>
  <div class="tab-pane" id="profile">...</div>
</div>

事件

事件 描述
show 激活一个选项卡且还没有执行其它操作时，可以使用 event.target 和 event.relatedTarget 进行一些操作
shown 激活一个选项卡之后，可以使用 event.target 和 event.relatedTarget 进行一些操作 
$('a[data-toggle="tab"]').on('shown', function (e) {
    e.target // 当前
    e.relatedTarget // 前一个
})

携程页面选择

实例

点击触发 
$('#spage02').querypage({
    later: function(data){
        $('#spage02').html(data.name?data.name:data.id)
    }
})

用法

启动实例：

$("#demo").querypage({
    // 以下配置
})

配置

配置项 类型 初始值 说明
position absolute/fixed absolute 日历定位方式
container string body 提示信息Dom插入位置，默认为body下面，可设置显示在任意位置，如： container: '#demoxx'
zindex number 1100 日历层级
later null | function null 选择页面时执行操作，callback为页面相关信息json
url.channel string (url) http://svn.ui.sh.ctripcorp.com/p/getchannels.php?ismain=1&callback=querycpage.channel 取频道数据
url.mainpage string (url) http://svn.ui.sh.ctripcorp.com/p/getpages.php?ismain=1&callback=querycpage.page 取常用主流程页面数据
url.getchannelpage string (url) http://svn.ui.sh.ctripcorp.com/p/getpages.php?cid={keyword}&callback=querycpage.page 由频道取页面数据
url.getpage string (url) http://svn.ui.sh.ctripcorp.com/p/getpageinfo.php?key={keyword}&callback=querycpage.page 由pagename或pageid取页面数据

用法

$('#example').tooltip(options)

配置项可以是HTML属性，也可JS设置，如： data-animation="true" 或 animation: true
HTML属性需要添加 data- 头 ，如 data-animation=""

配置

配置项 类型 初始值 说明
animation boolean true 是否使用css fade动画
html boolean false 提示内容是否解析HTML，默认以文本方式解析
placement string | function 'top' 显示位置 - top | bottom | left | right
selector string false 指定Target下的操作目标，默认是Target本身
title string | function '' 显示内容，如果不存在则取标签 title 内容
trigger string 'hover focus' 提示操作方式 - click | hover | focus | manual
delay number | object 0 提示信息延迟时间，可分别设置显示和隐藏延迟，如下：delay: { show: 500, hide: 100 }
container string | false false 提示信息Dom插入位置，默认为Target下面，可设置显示在任意位置，如：container: 'body'

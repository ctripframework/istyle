Loading

样式


用法

启动实例：

$("#loaddemo").loading()

关闭实例：

$("#loaddemo").loading('hide')

配置

配置项可以是HTML属性，也可JS设置，如： data-mask="true" 或 mask: true

配置项 类型 初始值 说明
show boolean true 默认执行实例show方法
mask boolean true 是否显示遮罩层
fadeout boolean false 关机实例时是否动画
zindex number 1000 实例层级
offsetTop number 0 实例模块top偏移值
offsetLeft number 0 实例模块left偏移值
icon boolean | string true true时显示图标；false时不显示图标；
string时表示更改图标url
style css '' 附加或者重置样式
maskOffset top 0 遮罩层top值
left 0 遮罩层left值
right 0 遮罩层right值
bottom 0 遮罩层bottom值
position absolute | fixed absolute 实例定位方式
text string | html 数据加载中... 显示文本

方法

$().loading()

启动实例 

$().loading()

关闭实例

$().loading('hide')
